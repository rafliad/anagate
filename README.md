# Anagate Landing Page

Public Astro landing page for Anagate, a pre-signing policy gateway for AI-agent payments on Stellar.

## Commands

```bash
npm.cmd run dev
npm.cmd run check
npm.cmd run build
npm.cmd run preview
```

## Vercel

Use the default static Astro setup.

- Build command: `npm run build`
- Output directory: `dist`
- Framework preset: Astro
